# 🌗 Tema Dark/Light

Projeto simples desenvolvido com HTML, CSS e JavaScript que permite alternar entre tema claro e escuro.

## 🚀 Funcionalidades

- Alternância entre tema claro e escuro
- Salvamento da preferência do usuário (localStorage)
- Detecção automática do tema do sistema
- Transição suave entre cores
- Botão com ícones 🌙 / ☀️

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (variáveis CSS)
- JavaScript (DOM e eventos)


## 💡 Aprendizados

Neste projeto pratiquei:
- Manipulação do DOM
- Uso de `classList`
- Eventos com JavaScript
- Uso de `localStorage`
- Organização de código
